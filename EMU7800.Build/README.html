EMU7800 Build README

Install Prerequisites

  • Visual Studio 2013 + latest update(s)
    • EMU7800.D2D\EMU7800.D2D.Win32.sln - Direct2D Win32
    • EMU7800.D2D\EMU7800.D2D.WinRT.sln - Direct2D Windows Store 8
    • EMU7800.D2D\EMU7800.D2D.WinRT_8.1.sln Direct2D Windows Store 8.1
    • EMU7800.D2D\EMU7800.D2D.WP.sln - Not used, but intended for Windows Phone should Direct2D appear on the platform
    • EMU7800.WP\EMU7800.WP8.sln - Windows Phone 8
    • EMU7800.Win\EMU7800.sln - Classic Win32
    • EMU7800.SL\EMU7800.SL.xln - Silverlight 5
  • Visual Studio 2012 + latest update(s)
    • EMU7800.WP\EMU7800.WP.VS2012.sln - Windows Phone 7
  • Silverlight 5 SDK
  • Windows Phone 8.0 SDK

Functions

  • BUILDALL.cmd - Builds everything (excluding Windows Phone and Windows Store editions):
    • Binaries: Debug/Release/ARM: Win, SL, WP, D2D, DumpBin
    • .zip archives of binaries where applicable
    • .zip source code archive
  • BUILDDROPALL.cmd - Prepares binary distributables (i.e., DROP\ folder), but does not gather source distribution.
  • CLEAN.cmd - Cleans everything
