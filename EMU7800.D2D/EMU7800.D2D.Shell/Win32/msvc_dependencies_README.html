Microsoft Visual C++ Runtime Library Dependencies

In this .csproj file, an ItemGroup has been conditionalized by build $(Configuration) and parameterized by build $(Configuration) and $(Platform) to reference the following dependencies: msvcp110.dll, msvcp110d.dll, msvcr110.dll, msvcr110d.dll.

While Solution Explorer annotates these files with a yellow triangle, the delivery of the right modules for the right build $(Configuration) and $(Platform) works without warnings. Further, the ClickOnce Publish dialog correctly recognizes these prerequsites.
